# Content Tree Application

## Description

This is a take home assignment for a senior java developer role.

## Run the project

### Prerequisites

1. Java 25
2. Maven 3.9.x+
3. Node 22.x+

### Backend

Run these commands from the project root:

```
mvn spring-boot:run
```

### Frontend

Cd into frontend folder and run:

```
cd frontend
```

```
npm install
```

```
ng serve
```
